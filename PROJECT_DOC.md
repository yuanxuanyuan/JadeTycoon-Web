# 翡翠大亨 (Jade Tycoon) - 项目文档

## 一、项目概述

**翡翠大亨** 是一款翡翠经营模拟网页游戏，采用 Roguelike 风格，玩家从路边摊起步，通过买卖原石、切割、雕刻、直播、经营实体商铺等方式积累财富，应对债务压力，最终成为翡翠大亨。

- **技术栈**：React 19 + Vite 7 + Tailwind CSS
- **类型**：单页应用 (SPA)，纯前端，存档存于 localStorage

## 三、核心玩法流程

```
开局选择剧本 → 市场买原石 → 工作台切割 → 多种变现路径
                    ↓
    ┌───────────────┼───────────────┬───────────────┬───────────────┐
    │               │               │               │               │
 直接售出      找 NPC 议价      送雕刻师      直播售卖      实体商铺上架
    │               │               │               │               │
    └───────────────┴───────────────┴───────────────┴───────────────┘
                                    ↓
                            雕刻成品进背包 → 售出 / 直播卖 / 送福利 / 上架商铺
```

### 时间系统
- 每 N 次操作 = 1 天（默认 3，遗物「挑灯夜战油」+1）
- 每日推进：随机事件、收藏升值、私信、走私客、雕刻师取货等

## 五、功能模块详解

### 5.1 原石市场

- **5 个等级**：路边摊 → 小型毛料行 → 批发市场 → 公盘竞拍行 → 秘境矿脉
- **3 个渠道**：
  - **公斤料区**：按公斤计价的蒙头料，赌性最大
  - **精品柜台**：半明料，品质下限有保障，可砍价
  - **暗标公盘**：竞标购买
- **打灯**：消耗 ¥5000，揭示 hiddenTag，影响切割概率
- **场口**：帕敢、龙塘、后江、会卡、南齐、抹谷、其他，各有概率修正

### 5.2 切割工作台

- **切割结果**：砖头料 / 糯种 / 花青 / 冰种 / 玻璃种 / 帝王绿
- **半明料**：开窗后可按半明料估值售出
- **精品收藏**：冰种以上概率出精品，可收藏每日升值
- **盘货**：送雕刻师「盘货」，提升成品估值

### 5.3 NPC 人脉（24 位）

- **老王、小美、陈师傅、张总、肥叔、李教授** 等
- **好感等级**：路人 → 铁粉 → 舰长 → 神豪（交易次数升级）
- **出价逻辑**：baseValue × bonus × 相性加成 × 产品偏好
- **砖头料风险**：卖砖头料有被识破概率，掉好感

### 5.4 雕刻师（12 位）

- **华师傅、李天王、阿福、褚石翁、鬼手·阿九、巧娘·金燕** 等
- **专长**：手镯、平安扣、珠串、佛像、龙牌等
- **特殊机制**：褚石翁点石成金、鬼手灵光乍现、枯木禅师 3 天取货开光等
- **灵感 / 能量**：互动消耗，影响雕刻效果

### 5.5 直播间

- **5 级**：未开 → 初级 → 中级 → 高级 → 顶级
- **观众池**：50 人，有粉丝等级、钱包状态
- **售卖 / 盲拍**：切好的石头或饰品直播卖；未切原石可盲拍，成交后代客主刀
- **送福利**：直播中给观众送翡翠，提升好感
- **房管后台**：查看 50 人成分、状态

### 5.6 实体商铺

- **5 级**：破旧当铺 → 临街玉器行 → 品牌旗舰店 → 豪华珠宝楼 → 顶级会所
- **货架**：从背包选切割成品或饰品上架，自定义标价
- **访客**：100 位商铺 NPC（tourist / citizen / tycoon / expert / special）
- **行为**：逛 → 看货架 → 购买 / 砍价
- **开门条件**：资金 ≥ ¥300 万

### 5.7 私信系统

- **200 条消息模板**：借钱的、订单的、红包、八卦等
- **NPC 列表**：与市场 NPC 部分重叠
- **行为**：借钱、领红包、接订单（需背包有对应饰品）、走私客到货等
- **走私客**：转账后次日 40% 被拉黑 / 60% 收到极品原石

### 5.8 每日随机事件

- **15% 触发**：正面（原石打折、切割加成、NPC 出价加成）或负面（涨价、识破率提升等）
- **持续时间**：1–3 天
- **modifiers**：marketPrice、npcOffer、cutQuality、brickDetect 等

### 5.9 遗物（黑市）

- **触发**：每逢天数尾数为 5（5、15、25…）右下角出现黑市入口
- **30+ 种遗物**：紫光手电、镶金切刀、转运符、龟甲等
- **效果**：当局生效，影响切割、NPC、收藏、债务等

### 5.10 收藏馆

- **精品**：冰种以上切割有小概率出收藏品
- **每日升值**：持有天数越长价值越高
- **李教授 Lv2**：藏馆升值 +30%

## 六、数据与配置

### 6.1 原石品质
- common / uncommon / rare / epic / legendary
- 影响可切割出的种水上限

### 6.2 切割结果（CUT_RESULTS）
| id | 名称 | 倍率 | 基础概率 |
|----|------|------|----------|
| brick | 砖头料 | 0.4x | 40% |
| waxy | 糯种 | 2x | 32% |
| flower | 花青 | 3.5x | 13% |
| ice | 冰种 | 6x | 11% |
| glass | 玻璃种 | 12x | 3% |
| imperial | 帝王绿 | 24x | 1% |

### 6.3 饰品档位（JADE_GRADES）
白档 / 绿档 / 蓝档 / 紫档 / 橙档 / 红档，对应不同估值倍率与颜色

### 6.4 商铺等级（SHOP_LEVELS）
- 货架数：3 / 5 / 8 / 12 / 20
- 每日客流上限：5 / 12 / 25 / 45 / 80

## 七、模块间联系

```
                    ┌──────────────────┐
                    │   App.jsx 主状态   │
                    └────────┬─────────┘
                             │
     ┌───────────────────────┼───────────────────────┐
     │                       │                       │
     ▼                       ▼                       ▼
┌─────────┐            ┌──────────┐            ┌──────────────┐
│ 市场    │ 购买 →     │ 工作台   │ 切割 →     │ inventory    │
│ (市场等级)│            │ (切割/盘货)│            │ 石头列表    │
└─────────┘            └────┬─────┘            └──────┬───────┘
                            │                         │
              ┌─────────────┼─────────────┐           │
              ▼             ▼             ▼           │
        ┌──────────┐ ┌──────────┐ ┌──────────┐       │
        │ NPC 议价  │ │ 雕刻师   │ │ 直播售卖  │       │
        └──────────┘ └────┬─────┘ └──────────┘       │
                          │                           │
                          ▼                           ▼
                    ┌──────────────┐            ┌──────────────┐
                    │jadeAccessories│            │  ShopPanel   │
                    │ 饰品背包      │ 上架 →     │ 商铺货架     │
                    └──────┬───────┘            └──────────────┘
                           │
                           ├→ 直接售出
                           ├→ 找 NPC
                           ├→ 直播卖
                           └→ 送福利
```

### 关键数据流
- **inventory**：切割石头，来自市场购买
- **jadeAccessories**：雕刻成品，来自雕刻师
- **shopShelves**：商铺上架商品，来自 inventory（石头）或 jadeAccessories（饰品）
- **npcRelations**：与 24 位 NPC 的交易次数，影响出价与技能解锁
- **phoneMessages**：私信记录，可触发订单、借钱、红包、走私客等
